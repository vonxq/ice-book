# Ice Book - 智能记账应用

一款面向个人用户的智能记账应用，帮助用户轻松管理日常收支、资产配置，并通过数据可视化提供财务洞察。

## 🚀 功能特性

### 核心功能
- **简单记账** - 3步完成记账，操作简单直观
- **智能分类** - 自动识别交易类型，减少手动分类
- **资产全景** - 统一管理所有账户资产，实时掌握财务状况
- **数据洞察** - 通过图表分析消费习惯，助力理性消费

### 主要模块
- **主页** - 财务概览和工具入口
- **记账** - 快速记账功能，支持收入/支出记录
- **我的小家** - 家庭成员管理和零花钱功能
- **资产管家** - 资产管理和图表分析
- **图表** - 数据统计和分析

## 📱 界面设计

### 设计风格
- **设计语言** - iOS原生风格（Cupertino Design）
- **色彩方案** - 支持多套主题配色，支持深色/浅色模式切换
- **字体** - iOS系统字体（SF Pro Display），支持动态字体大小
- **图标** - 字体图标，无图片资源依赖

### 主题配色
1. **晨曦紫** - 温暖而优雅的紫色调
2. **海洋蓝** - 清新自然的蓝色调
3. **森林绿** - 生机勃勃的绿色调
4. **日落橙** - 温暖活力的橙色调
5. **极简灰** - 经典简约的灰色调

## 🛠️ 技术栈

- **框架** - Flutter 3.0+
- **语言** - Dart
- **状态管理** - Provider
- **本地存储** - SQLite (sqflite)
- **图表库** - fl_chart
- **字体图标** - font_awesome_flutter

## 📦 安装和运行

### 环境要求
- Flutter SDK 3.0+
- Dart SDK 3.0+
- iOS 12.0+ / Android 8.0+

### 快速开始

1. **克隆项目**
```bash
git clone <repository-url>
cd ice-book
```

2. **安装依赖**
```bash
flutter pub get
```

3. **运行应用**

#### 方式一：平台选择启动器（推荐）
```bash
./start.sh
```
然后选择运行平台：
- macOS 桌面应用
- Web 浏览器应用
- iOS 模拟器
- Android 模拟器
- 自动选择最佳平台

#### 方式二：直接运行
```bash
# macOS 桌面应用
flutter run -d macos

# Web 浏览器应用
flutter run -d chrome

# iOS 模拟器
flutter run -d "iPhone 16 Pro"
flutter run -d "iPhone 16"
flutter run -d "iPhone"

# Android 模拟器
flutter run -d "android"
```

#### 方式三：iOS专用脚本
```bash
# iOS版本快速启动
./run_ios.sh
```

#### 方式三：使用运行脚本
```bash
# 自动选择平台
./run.sh

# Web版本（需要Python）
./run_web.sh
```

### 开发模式

```bash
# 生成代码
flutter packages pub run build_runner build --delete-conflicting-outputs

# 热重载开发
flutter run --hot
```

## 📊 数据模型

### 核心实体

## 🎯 用户流程

### 新用户引导
1. 应用启动 → 欢迎页面
2. 功能介绍 → 功能亮点展示
3. 权限申请 → 通知权限、存储权限
4. 初始设置 → 添加第一个账户
5. 完成引导 → 进入记账主页

### 记账流程
1. 点击记账按钮
2. 选择收入/支出类型
3. 选择分类
4. 输入金额（计算器界面）
5. 填写备注和日期
6. 确认保存
